<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Summary recomputes decisions and diverges from reviewed preview state | High | Aggregator reads existing JSON state only and has tests that monkeypatch policy/provider paths out of scope. |
| Partial workflow state makes summary unusable | Medium | Missing workflows are recorded in `missing_workflows` and rendered in Markdown. |
| Summary command mutates GitHub | High | CLI only reads/writes local state and prints preview safety text; mutation scan remains in Phase 4. |
| Summary state grows unbounded | Medium | One canonical `summary/records.json` and one overwritten `latest-preview.md` are written per run. |
</threat_model>
