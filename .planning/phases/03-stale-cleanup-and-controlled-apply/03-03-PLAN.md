<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Apply performs a mutation for an unreviewed issue | High | Require an existing matching close preview record and reject missing/mismatched records. |
| Apply recomputes closure policy and changes the reviewed action | High | Apply loads preview records only and does not call closure policy. |
| A label/comment succeeds but close fails, hiding partial state | High | Record separate action statuses and preserve errors without retry loops. |
| Tests accidentally mutate GitHub | High | Apply tests use fake runners; CLI apply tests avoid live `gh` execution. |
| Missing labels are created implicitly | Medium | Adapter may add/remove existing labels only; no label creation command is introduced. |
</threat_model>
