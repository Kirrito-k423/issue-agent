## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Closure policy treats old age as evidence | High | This plan only creates linked evidence and fixture signals; policy must later require explicit closure reason evidence. |
| PR state is guessed from URL text | High | Unknown status is represented explicitly and is not considered closure evidence. |
| Cross-repo references are parsed into unsafe local assumptions | Medium | Store repository separately and do not treat cross-repo refs as resolved unless status is known. |
| Comment content injection changes command behavior | Medium | This plan only parses text into models; no shell or GitHub mutation commands are introduced. |
</threat_model>
