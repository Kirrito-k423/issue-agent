<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Old issues are recommended for closure without evidence | High | Closure policy requires allowed reason evidence and tests old-but-not-closable fixtures. |
| Roadmap/help-wanted/open-PR issues are closed by automation | High | Policy blocks these labels/signals unless explicit completion or supersession evidence exists. |
| Close preview mutates GitHub | High | CLI only writes local state and verification scans preview code for mutation commands. |
| Maintainer cannot audit why a closure is suggested | Medium | Preview records include risk level, reason, current state summary, evidence refs, and draft comment path/text. |
</threat_model>
